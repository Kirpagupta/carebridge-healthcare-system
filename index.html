🎯 What is CareBridge?

CareBridge is a Department Recommendation System that takes a patient's symptoms as input, runs them through 6 trained ML/DL models, predicts the most likely medical condition, and recommends the correct hospital department — saving time and ensuring patients reach the right specialist.

🔬 ML Pipeline

  • Dataset: itachi9604 Disease-Symptom Prediction Dataset — 42 diseases, 138 symptoms, ~4900 patient records
  • Feature engineering: Binary symptom matrix + severity score (sum of weights)
  • Noise augmentation: 3% bit-flip on binary features to prevent data leakage
  • Train/test split: 80/20 stratified by disease class
  • 6 classification models + 1 regression + 1 clustering (K-Means + PCA)
  • Best model selected by test accuracy; all others used for consensus view

📊 Statistical Analysis (FDA)

  • One-way ANOVA — severity scores differ significantly across diseases (p < 0.05)
  • Chi-square test — disease type and risk level are associated (p < 0.05)
  • Pearson Correlation — high-severity symptoms identified as strong predictors
  • Linear Regression — R² = 0.89 for severity score prediction
